Css Nedir?

Css nin açılımı cascading sytle sheet dir.Ekrandaki Html ögelerinin tasarımını yapmamızı sağlar.1 tane css dosyasıyla tüm uygulamayı dizayn edebildiğimiz gibi 1 den çok stil dosyası da kullanabiliriz.

Not:Herhangi bir Css uygulamazsak her tarayıcı kendi cs kodlarıyla html elementlerimizi şekillendirir.Dolayısıyla chrome da düzgün görünen bir tasarım firefox tarayıcı da farklı görünebilir!Buna tarayıcı uyumsuzluğu denir.Bu sorunu çözmek için genelde bir css dosyasıyla tüm elementlerin css leri sıfırlanıp spnrasında kendi kurallarımızı ekleriz.

Ayrıca uygulamalarda gerektiğinde harici bir eklenti kullanmamız gerekebilir.Kullandığımız eklentinin de kendine ait tasarım dosyası olabilir bunu da projeye dahil ederek kullanırız

CSS Yazma Seçenekleri

  1. Inline yani direkt html elemntinin içine style attribute ü ile yazabiliriz.
  2. Sayfada head kodları arasında style tag larının arasına yazabiliriz
  3. Harici bir dosya oluşturup (still.css gibi) bu dosyayı yine sayfanın head alanına çağırıp tüm sayfaları 1 dosya ile dizayn edebiliriz

Css uygulamak için öncelikle css uygulayacağımız element ya da elementleri seçip sonrasında css kodları ile bu elementlere css uygulayabiliriz.

2-Inlıne Css Yazımı

Elementin başlangıç tag leri arasına style="" diyerek css yazabiliriz Eğer bir den çok css özelliği ekleyeceksek aralarına ; ekleyerek yazarız.

Harici Css Dosyası Kullanma

Projeye (veya projeki istediğimiz bir klasöre)sağ tıklayıp add > Cascading Sytle sheet menusunden oluşacak css dosyasına dilediğimiz ismi veri ok diyerek .css uzantısıyla dosyayı oluşturuyoruz.

Oluşan css dosyası içerisine sadece css kodları yazıyoruz.

Son olarak oluşturduğumuz css dosyasının sayfamızda çalışması için sayfanın head kodları arasına dosyayı sürükle bırak yöntemi ile veya elle link kodu yazarak dahil etmeliyiz aksi takdirde yazacağımız css kodları sayfaya işlemez

Not: Harici css kullandığımızda bazen yaptığımız css değişiklikleri tarayıcıda uygulanmaz bunun sebebi tarayıcıların css gibi statik dosyaları ön belleklerine almalarıdır. Bu durumda klavyeden ctrl tuşuna basılı tutarak sayfayı refresh yaparsak css dosyası yeniden kaynağından çekilir ve ön bellek boşaltılarak bu yeni dosya kullanılır.

CSS Seçiciler

1-Element Seçici

Sayfadaki Html elemntleri seçerek bu elementlere css uygulayabiliriz. Örneğin body elementi veya div,p,h1-h6,ul vb gibi.

2-Id Seçici

Elementlere sonradan tanımlayacağımız id attribute ü ile ulaşıp css kodu uygulayabiiriz. Bir elemente id si ile ulaşmak için css yazdığımız yerde # işareti koyup id ye yazdığımız değeri belirtiriz,şöyle;#idSecici{buraya css kodları yazılır}

3-Class Seçici

Css class seçici kullanabilmek için de herhangi bir html elemntinin başlangıç etiket isminden sonra 1 boşluk bırakıp class ="" şeklinde tanımlama yaparız. İstediğimiz class ismini verebiliriz. Css tarafında class seçici ile kod yazmak için . seçicisi kullanılır. Style kodları arasında .classSecici{}şeklinde lementlere ulaşıp css uygulayabiliriz.

Not: id seçiciyi genellikle 1 sayfadaki sadece 1 element için kullanırız! Class seçiciyi ise birden çok elementi seçmek için kullanıırız.
